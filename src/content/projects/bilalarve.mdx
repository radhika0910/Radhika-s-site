## Bilal Arve's Personal Site
Dive into the digital world through the lens of Bilal Arve, a talented UI/UX designer hailing from Banda Aceh, Indonesia. His personal website, built with Next.js and TailwindCSS, is a testament to the seamless integration of technology and creativity, enhanced with captivating Lottie animations.
[Preview](https://www.bilalarve.com/)


### Features
- **Animation Delight:** Lottie animations elevate the visual appeal, bringing a dynamic touch to the website.
- **Responsive Design:** Enjoy a seamless experience across devices with a responsive design tailored for mobile accessibility.
- **Accessibility:** Prioritizing inclusivity, the website is designed to work seamlessly with screen readers.

### Stack
- [Next](https://nextjs.org) - Next.js is a React framework for building full-stack web applications
- [TailwindCSS](https://tailwindcss.com) - Tailwind CSS: A utility-first CSS framework offering design flexibility.
- [Lottie](https://lottiefiles.com/) - Lottie: A JSON-based animation format enriching the website with captivating animations.
